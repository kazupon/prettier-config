# @kazupon/prettier-config

Prettier config for @kauzpon

## 🚀 Usages

### 💿 Installation

```sh
# Using npm
npm install --save-dev @kazupon/prettier-config

# Using pnpm
pnpm add -D @kazupon/prettier-config

# Using yarn
yarn add --dev @kazupon/prettier-config
```

### ✏️ Edit `package.json`

```json
{
  // ...
  "prettier": "@kazupon/prettier-config"
}
```
